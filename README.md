# CineScope 🎬 — 电影字幕分析工具

**纯前端** SRT 字幕分析工具，上传字幕文件即可自动生成多维度可视化报告。数据仅在你浏览器中处理，不上传任何服务器。

![CineScope](https://img.shields.io/badge/CineScope-v1.0-gold)
![Chart.js](https://img.shields.io/badge/Chart.js-4.4.1-blue)
![纯前端](https://img.shields.io/badge/纯前端-✓-brightgreen)

---

## 功能

| 图表 | 类型 | 说明 |
|------|------|------|
| 🎭 角色台词量 | 柱状图 + 饼图 | 每个角色的台词条数及占比 |
| 💗 情感曲线 | 折线图 + 堆叠面积图 | 逐条字幕情感值变化 + 正/中/负面分布 |
| ☁️ 词云 | Canvas 词云 | 高频关键词可视化，基于中文情感词典 |
| 🎬 场景切换 | 柱状图 + 折线 | 场景台词密度与持续时间 |

### 其他特性

- 暗色电影主题 UI，金色点缀
- 拖拽上传 `.srt` 文件
- 示例数据一键加载（《给阿嬷的情书》）
- 自适应布局，支持移动端
- 统计概览卡片（台词数、字数、角色数、场景数、情感倾向、时长）

## 使用

### 在线使用

直接用浏览器打开 `index.html` 即可，无需构建工具，无需安装。

### 本地运行

1. 克隆或下载本项目
2. 使用任意 HTTP 服务器托管（因 `fetch` 加载示例数据需要 HTTP 环境）：

```bash
# 使用 Python
python3 -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 VS Code Live Server 扩展
```

3. 浏览器打开 `http://localhost:8000`

### 使用方式

1. **上传字幕** — 拖拽 `.srt` 文件到上传区域，或点击浏览选择
2. **加载示例** — 点击"加载示例数据"体验《给阿嬷的情书》分析
3. **点击分析** — 点击"分析字幕"按钮即可生成所有图表

## 技术栈

- **HTML5 / CSS3** — 原生，无框架
- **Chart.js 4.4.1** — 图表渲染（CDN 加载）
- **Canvas API** — 词云绘制（螺旋碰撞检测算法）
- **纯原生 JS** — 所有逻辑零依赖

## 开发

### 情感词典

`js/analyzer.js` 内置了面向中文电影台词的情感词典：

- **正面词汇**：爱、喜欢、幸福、开心、感动、温暖 … (约 80 词)
- **负面词汇**：哭、悲伤、难过、痛苦、遗憾 … (约 70 词)
- 支持情感放大器（很、非常、太、好）
- 支持标点情感影响（！、？、…）

### 场景检测算法

- **时间间隔法**：相邻字幕间隔 > 2 秒 → 新场景
- **关键词法**：检测"第二天"、"几个月后"、"回忆"等场景切换关键词
- 融合两种方法的结果

### 词云

Canvas 词云使用 Archimedean 螺旋放置算法，支持中英文混排，最大展示 80 个高频词，字号按词频比例缩放。
